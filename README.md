# EasyVideoRecorder #

EasyVideoRecorder是EasyDarwin开源团队成员开发的一款移动端拍摄库，类似于美拍、秒拍、微拍的短视频拍摄功能，支持按住拍、滤镜、mv、照片合成、水印、本地视频导入功能，并且还在不断扩展功能中！支持Android、IOS平台，适配性好，比同行业的视频合成技术在：速度，稳定性，适配性上都具有一定的技术优势！

**EasyPublisher SDK库个人使用免费，企业及商用需要经过授权，支持滤镜、MV定制**；
